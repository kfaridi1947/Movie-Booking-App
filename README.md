# 🎬 Movie Booking App

A modern full-stack Movie Booking Application built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). This application allows users to browse movies, view details, select seats, and book tickets through an intuitive and responsive interface.

## 🚀 Features

* User Authentication (Register/Login)
* Browse Available Movies
* View Movie Details
* Seat Selection System
* Ticket Booking Management
* Responsive User Interface
* Protected Routes
* RESTful API Integration
* MongoDB Database Storage

## ⚡ Installation

### Clone Repository

```bash
git clone https://github.com/kfaridi1947/Movie-Booking-App.git
cd Movie-Booking-App
```

### Install Backend Dependencies

```bash
cd server
npm install
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

## 🔧 Environment Variables

Create a `.env` file inside the server directory:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

## ▶️ Run Application

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
cd client
npm start
```

Application will run on:

```bash
http://localhost:3000
```
